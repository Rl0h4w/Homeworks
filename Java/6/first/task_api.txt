### Создать задачу
POST /api/tasks
{
    "title": "твоё значение", // Заголовок задачи
    "description": "твоё значение", // Описание задачи
    "status": "твоё значение",  // Возможные значения: "pending", "in_progress", "completed"
    "dueDate": "твоё значение" // Формат: "YYYY-MM-DD"
}

Returns: 201 Created
{
    "id": "твоё значение", // Уникальный идентификатор задачи

    "status": "твоё значение",
    "dueDate": "твоё значение"
